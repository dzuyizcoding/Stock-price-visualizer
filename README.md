📈 Stock Price Visualizer

A Python tool that fetches and visualizes recent stock price data for Hong Kong blue-chip companies. The program retrieves historical market data, processes it using Pandas, and generates multi-panel charts showing price trends over the last 10 trading days.

🚀 Features

Fetches real market data using yfinance

Processes stock data using Pandas

Automatically exports data to CSV

Generates multi-panel price charts with Matplotlib

Visualizes 10 most recent trading days

Includes error handling and logging for reliability

🛠 Tech Stack

Python

Pandas

Matplotlib

yFinance API

📊 Example Output

The program generates:

stock_prices.csv → structured dataset

stock_prices.png → visualized stock charts
